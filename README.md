# 📇 Contact Management System

A full-stack web application built using React.js for the frontend and Django REST Framework for the backend. The application allows users to submit contact information through a responsive interface and store the data in a MySQL database.

## Features

- Contact Form Submission
- React Frontend
- Django Backend
- REST API Integration
- MySQL Database
- Responsive User Interface

## Technologies Used

### Frontend
- React.js
- JavaScript
- HTML5
- CSS3

### Backend
- Python
- Django
- Django REST Framework

### Database
- MySQL
